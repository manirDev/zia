## Syntaxe de base
Dans Zia, la syntaxe suit de près celle du langage C, mais avec des mots-clés en français. Voici quelques éléments de base :

---

## Mots cles

| French Keyword | Precedence     | Meaning (English Equivalent) |
| -------------- | -------------- | ---------------------------- |
| `et`           |                | and                          |
| `pour`         |                | for                          |
| `nul`          |                | null                         |
| `ou`           |                | or                           |
| `afficher`     |                | print                        |
| `retourner`    |                | return                       |
| `tantque`      |                | while                        |
| `classe`       |                | class                        |
| `ceci`         |                | this                         |
| `vrai`         |                | true                         |
| `var`          |                | var                          |
| `fonction`     |                | function                     |
| `faux`         |                | false                        |
| `super`        |                | super                        |
| `si`           |                | if                           |
| `sinon`        |                | else                         |

----
